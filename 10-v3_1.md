# DD API V3.1

DD API V3.1 is een optionele uitbreiding op DD API V3 en bevat een tweetal specificaties, beiden optioneel:
 
- [Pub/Sub](11-pubsub.md) (publish/subscribe) op wijzigingen in observaties data. __PubSub__ is __niet__ READ-ONLY.
- [Zoek-profielen](12-search-profiles.md) voor vooraf-opgestelde selecties (`$filter` en `$select`) van

Deze zijn optioneel, omdat ze niet door alle implementaties ondersteund hoeven te worden, wegens specifieke use-cases of technische beperkingen.
